# 末日背包 - 生存背包管理系统

## 项目简介
一款基于 Unity 的背包管理原型，参考《阿瑞斯病毒》的核心设计理念，模拟生存游戏中的资源取舍与策略决策。

## 功能亮点
- 左键点击拾取物品，右键丢弃物品
- 拖拽交换/移动物品
- 一键整理（合并同类物品，按名称排序）
- 一键清空背包
- 负重上限系统（满负重无法拾取）
- 本地存档（关闭重开后状态保留）

## 技术栈
- Unity 2022.3 (2D URP)
- C#、UGUI、ScriptableObject、PlayerPrefs
- 拖拽接口 (IBeginDragHandler, IDropHandler)

## 操作说明
- 左键点击格子：拾取/增加物品
- 右键点击格子：丢弃物品
- 拖拽格子：交换物品位置
- 点击“整理”按钮：合并同类物品
- 点击“清空”按钮：清除所有物品
